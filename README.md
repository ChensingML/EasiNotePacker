# EasiNotePacker
一个简单的EasiNote5课件获取器，或者说是用户数据打包器

## 环境要求
1. Windows 操作系统
    + _至少_ 需要 Windows XP
    + x86 _或者_ x64 操作系统
2. 4GB 的 RAM

## 用法
1. 打开它!
2. 点击"搜索"!
3. 搜罗下有什么是你想要的!
4. 点击"获取"!
5. 选个地方来保存塔门!
6. 尽情享受吧!

## 软件如何工作
当~~老师~~用户从EasiNote服务器上下载文件时，EasiNote将会在 __C:\Users\\(User)\AppData\Roaming\Seewo\EasiNote5\Data__ 留下缓存. 程序将会收集这些缓存，并将它们打包成.enbx文件.
有关详细信息，请阅读代码!
